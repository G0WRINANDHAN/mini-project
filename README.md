# Simple Calculator in C

This is a basic calculator program written in C that performs arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Supports +, -, *, /
- Takes user input
- Uses switch-case logic
- Simple and beginner-friendly

## Technologies Used
- C Programming

## Sample Output

Enter an operator (+, -, *, /): +
Enter two numbers: 10 5
Result = 15

Enter an operator (+, -, *, /): *
Enter two numbers: 4 3
Result = 12
